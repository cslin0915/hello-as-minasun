## 致愚者們 😀

- Public? Private?
	- Private repo. 在**協作者**的設定上, 只能同時也是**管理者**
	- Public repo. 則可以透過 **Fork** 的方式，避免任意人修改 Repository 的檔案
- Free? Paid?
	- **Pro** 方案可以有**無限的**協作者
	- **Team** 方案可以有 **Access controls**
- eLand Wiki? Heaven? SVN (RD 用的版控軟體)?
